Defense Layers
L1 INPUT SECURITY
Azure Prompt Shields · Content Filters · Blocklist pattern matching. Blocks malicious prompts before the LLM ever sees them.
L2 OUTPUT SECURITY
Azure Language PII detection · Hate speech · Self-harm content filters. Scrubs agent responses before they reach the user.
C2 MELON DETECTOR
Contrastive injection detector. Runs a clean vs. poisoned tool output comparison to catch indirect prompt injection mid-loop.
C3 TOOL FIREWALL
Validates every tool call and tool output. Blocks unauthorized shell commands, HTTP exfiltration attempts, and SQL injection.
C4 APPROVAL WORKFLOW
Human-in-the-loop gate for high-risk actions. Holds the agent until an operator approves or rejects the pending operation.
